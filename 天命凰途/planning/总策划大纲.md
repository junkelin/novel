## 二、完整世界观设定

### 2.1 世界背景

- **时代**：现代都市（平行时空2024年），科技发达但玄学隐秘传承
- **地理**：以江城（虚构一线城市）为核心舞台，辐射全国及海外
- **社会结构**：
  - 表面：现代文明社会，科学主导
  - 隐层：**玄学界**真实存在——风水师、相师、符箓师、占卜师等以不同形式活跃于上流社会
  - 玄学界的规则：不公开暴露于大众视野，但在政商界、娱乐圈、富豪圈中拥有巨大隐性影响力
  - **"天机局"**：国家设立的玄学事务管理机构（半官方），负责协调国内玄学从业者，处理超自然事件

### 2.2 力量体系（玄学境界九品制）

主角的玄学修为分为**九品十八境**，每品分初境/巅峰两阶：

| 品级 | 称号 | 核心能力 | 突破关键节点 |
|------|------|---------|-------------|
| **九品** | 感灵者 | 能感知气场波动，偶尔灵光一闪 | 第一次主动感知到异常气息 |
| **八品** | 见微者 | 开阴阳眼（基础版），可见残影和气场颜色 | 系统激活后第一次看到灵体/气场 |
| **七品** | 辨相师 | 面相手相精准断事，准确率70%+ | 第一次公开算命且全中，建立信心 |
| **六品** | 风水士 | 堂局堪舆，布简单阵法，改小运势 | 成功改造一处风水局（如宿舍/店铺） |
| **五品** | 天机师 | 占卜推演，预知短期因果，画符驱邪 | 用占卜避开一次大灾/预知一场重大事件 |
| **四品** | 化煞宗师 | 强力阵法，镇煞化劫，改人大命格 | 救治一名被严重冲撞的人/破解复杂凶局 |
| **三品** | 通玄地师 | 因果推演（中长程），大型法阵，触摸天机 | 揭穿一场涉及多人命运的巨大阴谋 |
| **二品** | 国手天师 | 国宝级人物，一言可影响区域气运 | 被天机局/高层正式认可，参与国家级事件 |
| **一品** | 天命圣人 | 窥天道，逆天改命（有限度），立言立说 | 全书终极成就 |

**系统辅助机制**：
- 系统名：**「天命罗盘」**（外观：脑海中一枚古朴铜罗盘）
- 核心功能：
  - **望气**：看人/地/物的气运颜色（红=吉/旺，黑=凶/灾，紫=贵，灰=死）
  - **卜筮**：每日免费卜卦3次（可消耗积分增加），准确率随品级提升
  - **鉴宝**：识别古董真伪与价值
  - **任务链**：发布主线/支线任务，完成获「天命值」
  - **商城**：天命值兑换：玄学典籍、法器材料、特殊技能、现金资产
  - **成就系统**：达成特定成就解锁隐藏功能和剧情线

### 2.3 核心规则（世界观底层逻辑）

1. **天道平衡律**：凡是大改他人命运，必须付出等价代价（消耗自身气运/寿数/天命值）。系统的作用是**降低代价**而非消除代价。
2. **天机不可尽泄**：每次泄露天机过多，会遭遇"天谴"（反噬），表现为短时间内运气骤降或遭遇意外。
3. **因果纠缠律**：算得越多，与被算者的因果越深。对恶人算命会沾染孽业。
4. **玄学传承律**：真正的玄学本领需要师承+悟性+实践三者结合，系统可以加速但不能跳级。
5. **隐世公约**：玄学从业者不得在公开场合（媒体/网络）过度展示能力，否则会被"上面"约谈。

### 2.4 主要势力

#### 势力一：林家（主角家族）
- **性质**：江城顶级商贾世家，主营地产+珠宝+文化投资
- **立场**：主角的后盾和根基，前期提供资源，中期卷入危机需主角拯救
- **与主角关系**：血脉至亲，核心支撑力量
- **关键人物**：林震海（祖父，家族掌舵人）、沈云漪（母亲）、林晚晴（堂姐）

#### 势力二：「天机局」（国家玄学管理机构）
- **性质**：半官方机构，隶属特殊部门，管理全国玄学从业者
- **立场**：中立偏秩序，维护玄学界秩序，招募人才
- **与主角关系**：前期远观→中期接触招揽→后期合作/隶属
- **关键人物**：顾寒州（天机局特派员，神秘强大）、叶老（天机局顾问，泰斗级人物）

#### 势力三：华夏玄学会（民间玄学联盟）
- **性质**：全国性玄学行业协会，组织评级考试和行业自律
- **立场**：维护行业利益，既竞争又排外
- **与主角关系**：初期轻视打压→被打脸后拉拢→后期臣服/合作
- **关键人物**：陈道陵（会长，伪君子，表面正派背后阴暗）、赵无极（副会长，直性子，后期成盟友）

#### 势力四：凤凰男阵营（前期反派）
- **性质**：以男主前世丈夫**周廷轩**为核心的利益集团
- **立场**：利用婚姻攀附权贵，谋夺家产
- **与主角关系**：前世仇敌，今生主角主动避坑+反向收割
- **关键人物**：周廷轩（凤凰男，伪善）、苏雨薇（前世小三，绿茶）、周母（极品婆母）

#### 势力五：暗网玄学黑市（中后期反派）
- **性质**：地下玄学交易市场，买卖诅咒/改命/邪术
- **立场**：唯利是图，不择手段
- **与主角关系**：中期的重大威胁来源，幕后黑手之一
- **关键人物**："鬼母"（黑市幕后操控者，神秘女人）

### 2.6 历史事件

**事件一：「九八年玄门大劫」（距今26年）**
- 一群野心玄学师试图用集体阵法改变区域国运牟取暴利
- 导致某大城市发生连环诡异事故，死伤众多
- 事后国家成立「天机局」强力整顿，玄学界转入地下
- 主角的祖父**林震海**是当年受害者之一（损失惨重但幸存），因此禁止家中后人接触玄学——这成为主角前期隐藏能力的理由

**事件二：「零八年汶川因果潮」（距今16年）**
- 大规模灾难导致区域性阴气暴涌，无数孤魂野鬼产生
- 全国玄学师联合超度，这是近代玄学界最大规模的公开协作
- 此事催生了新一代玄学传承者，其中就包括后来成为各方势力 key player 的人物
- 主角的前世在地震中做过志愿者（伏笔：她的灵魂因此变得更强韧，是重生后能承载系统的原因）

**事件三：「二二年天机泄漏事件」（距今2年）**
- 某网红"大师"在直播中展示了真实的玄术能力
- 短视频爆火后引发恐慌，被天机局迅速封杀
- 此事件导致玄学从业者进一步收紧自我约束，也促使天机局加强了新人筛查
- 这成为主角后期成名后需要小心的警示案例

## 三、全书总主线大纲（三幕式结构）

### 第一幕：凤还巢 —— 重生觉醒与初步立足（第1～120章，约25%）

**核心目标**：重生回归 → 觉醒系统 → 学好玄学 → 在校园和家族中站稳脚跟 → 初次打脸周边小人 → 避开周廷轩的陷阱 → 建立第一波口碑

**主要事件**：
1. **前世惨死与重生**：揭示前世结局（被周廷轩和苏雨薇设计，签下离婚+财产转移协议后被推下楼身亡），重生到大学报到日（18岁）
2. **系统觉醒**：发现天命罗盘，初次使用望气功能，看到周围人的气运颜色
3. **避开凤凰男**：周廷轩以"好心学长"身份接近，主角冷眼旁观，用系统看穿他表面热心实则贪婪的本质
4. **玄学启蒙**：在旧书摊/图书馆偶然获得基础玄学典籍（实为系统引导），开始自学
5. **第一次打脸**：帮舍友/同学解决小麻烦（如：看出某人男友出轨/某人有小人之相），初显能力
6. **家族风波**：林家商业合作中出现"意外"（有人做手脚），主角用风水眼光发现问题端倪并暗中提醒祖父
7. **第一桶金**：通过鉴宝/赌石/算命赚得第一笔钱（证明自己不只是花瓶富二代）
8. **结识关键盟友**：遇到生命中的重要伙伴（可能是未来的商业合伙人/玄学同道/天机局线人）
9. **周廷轩暴露真面目**：他开始追求主角的堂姐/闺蜜作为备胎计划B，主角暗中收集证据
10. **第一幕高潮——"江城名流宴斗法"**：林家受邀参加顶级商会宴席，对面阵营请来"大师"针对林家风水动手脚。主角当众揭穿对方诡计，以初出茅庐之姿一战成名。同时周廷轩在宴会上试图表现却弄巧成拙，彻底暴露其虚伪本质。

**主角状态变化**：迷茫新生 → 谨慎探索 → 建立自信 → 初露锋芒

**结尾钩子**：宴会上一位神秘老人（天机局的叶老）注意到主角，留下名片："小姑娘，你有天赋。有兴趣来聊聊？"

### 第二幕：龙抬头 —— 名声崛起与暗流汹涌（第121～360章）

**核心目标**：玄学实力飞速提升 → 名声从校园走向全国 → 商业帝国起步 → 与各方势力博弈 → 周廷轩阵营覆灭 → 卷入更大的阴谋 → 遭遇重大挫折

**矛盾升级节点（至少5次）**：

| 节点 | 章节（约） | 矛盾升级内容 |
|------|-----------|------------|
| **升级①** | 第130-150章 | 从校园走向社会，接第一个商业风水单子，面对真正业内人士的质疑和刁难 |
| **升级②** | 第170-190章 | 周廷轩勾结权势陷害林家，林家企业面临生死存亡，主角挺身而出力挽狂澜 |
| **升级③** | 第220-240章 | 华夏玄学会设局打压主角（因她未经过正规渠道入行），被迫参加全国玄学大赛打脸整个协会 |
| **升级④** | 第280-300章 | 发现前世死亡的真相不止周廷轩——背后还有更大势力（暗网黑市）在操纵一切 |
| **升级⑤** | 第330-350章 | 天机局内部出现内鬼，主角被陷害为泄露天机的罪人，遭到通缉/追杀 |

**第一次重大挫折（约第200章处）**：
- **事件**：主角因为一次算命泄露天机过多，触发天谴反噬——连续倒霉一周，关键时刻系统失灵，差点害得家人受伤
- **意义**：让她明白玄学的代价和规则，不是无所不能，学会谦逊和谨慎
- **走出挫折**：在最低谷时得到叶老指点，领悟"天道平衡"的真谛，突破瓶颈升级

**中期高潮（约第260章）——"周廷轩覆灭之战"**：
- 周廷轩已经靠骗娶豪门女积累了第一桶金和初步人脉（走的前世老路只是换了个对象）
- 主角收集齐全证据，精心布局，在他"人生巅峰时刻"（订婚宴/公司上市）当众揭露其所有罪行
- 苏雨薇倒戈（或被抛弃后反咬一口），周廷轩身败名裂，锒铛入狱
- **读者爽感峰值**：前世仇恨彻底清算！但——主角在清算过程中发现了周廷轩背后的"操盘手"，引出更大的阴谋

**黑暗最低谷（约第340-360章）**：
- **事件**：暗网玄学黑市浮出水面，主角发现他们一直在暗中监视自己
- 更可怕的是：**她的重生和系统可能并非偶然——有人在背后操控一切**
- 最信任的盟友之一背叛（或被要挟），主角陷入孤立无援
- 林家再次遭遇危机，这次比之前更严重（有人从风水层面直接破坏林家祖宅龙脉）
- 系统警告：检测到宿主命格受到外部力量压制，部分功能锁定

**主角状态变化**：意气风发 → 遭受打击 → 沉淀成长 → 绝地反击

**结尾钩子**：主角在祖宅地宫中发现一块古老石碑，上面的文字竟然和系统界面一模一样——**这个系统的来历，远比她想象的古老**

### 第三幕：凰临天下 —— 终极决战与天命归位（第361～500章）

**核心目标**：破解系统之谜 → 对抗暗网黑市 → 解救家族 → 登顶玄学界顶峰 → 逆转所有因果 → 最终结局

**最终反派揭示**：
- **表层反派**："鬼母"（暗网黑市操控者），企图用大规模邪术阵法攫取城市国运
- **深层反派**：天机局内部的**叛徒高层**（利用国家机器为自己谋私利，与黑市勾结）
- **终极真相**：主角的前世死亡、今生的系统觉醒，都是一场跨越两代的**天命棋局**——她的生父（已故）曾是天机局的核心成员，因为发现了叛徒的秘密而被灭口。系统是她父亲留下的最后保护机制。

**全书最高潮决战（约第430-450章）——"江城锁龙阵·终局之战"**：
- 鬼母联手叛徒高层启动"锁龙阵"，意图窃取整座城市的气运
- 主角集结所有盟友（林家、天机局忠贞派、玄学会正义方、甚至包括被策反的前敌人）
- 正面战场：玄学阵法对决（大规模风水大战）
- 暗线战场：主角利用系统终极功能——**"天命逆转"**——以自身一世修为为代价改写关键因果节点
- 决战高潮：主角突破至一品境界，天命罗盘完全体觉醒

**所有伏笔收线**：

| 伏笔方向 | 收线章节（约） | 收线方式 |
